# ThirdParty Recipe

## Add Submodule:googletest
```bash
cd path/to/SparseEnd2End
git submodule add https://github.com/google/googletest.git onboard/third_party/googletest
git commit - m "add submodule googletest"
cd  third_party/googletest
cmake -B build -S .
cmake --build build
```
